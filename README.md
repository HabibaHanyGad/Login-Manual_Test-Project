# Login-Manual_Test-Project
# This repository contains all Test Cases and Bug Reports for the Login functionality of the application, it is organized to make the testing process easy to follow and understand.

## Folder Structure

- **TestCases/** : Excel files with all test steps, data, and expected results
- **BugReports/** : Bugs found during testing, linked to related Test Cases
- **Attachments/** : Screenshots or logs for reference
  
## How to Use

1. Open the TestCases folder to review all test cases.
2. Execute tests in the specified environment (e.g., Windows 10, Chrome 118).
3. Check the BugReports folder for any issues found, and refer to Related Test Case column for traceability.
4. Refer to Attachments for screenshots if needed.
